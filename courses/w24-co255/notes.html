1 Overview

1.1 Optimization Problems

We can describe an optimization problem as a problem of the form minf(x)\min f(x) or maxf(x)\max f(x) where x∈Sx \in S, S⊆ℝnS \subseteq \mathbb{R}^n, and f:ℝn→ℝf: \mathbb{R}^n \to \mathbb{R}. The following terminology is used when refering to such problems:

  • ff is the objective function.
  • xx is the variable.
  • SS is the feasible region which contains feasible points and the feasible solution (if one exists).

Furthermore, our analysis can be restricted to one of either maxf(x)\max f(x) or minf(x)\min f(x) since both minf(x)\min f(x) and max−f(x)\max -f(x) are equivalent optimization problems (they have the same optimal solution and negated optimal values).

1.2 Possible Outcomes of an Optimization Problem

There are 4 possible outcomes of an optimization problem minf(x)\min f(x) for x∈Sx \in S, S⊆ℝnS \subseteq \mathbb{R}^n, and f:ℝn→ℝf: \mathbb{R}^n \to \mathbb{R}. They are:

  1. The problem is infeasible. This means that the feasible region, SS, is empty.
  2. There is an optimal value. We have that there is some optimal solution (not necessarily unique) x‾∈S\bar{x} \in S such that for all x∈Sx \in S, we have f(x)≥f(x‾)f(x) \ge f(\bar{x}).
  3. The problem is unbounded. This means that there are feasible solutions of arbitrarily small objective value.
  4. The problem is neither unbounded nor unfeasible. This means that inff(x)\inf f(x) is not attained on the feasible region.

1.3 Linear Programs

An optimization problem is linear when

  1. ff is an affine map.
  2. S={x∈ℝn:Ax≤b}S = \{ x \in \mathbb{R}^n : A x \le b \} where A∈ℝm×nA \in \mathbb{R}^{m \times n} and b∈ℝmb \in \mathbb{R}^m.

Note that here we say that given two vectors u,v∈ℝnu, v \in \mathbb{R}^n, we have that u≤vu \le v if and only if every component of uu is less than the corresponding component of vv.

In other words, a linear program is an optimization problem of the form minc⊤x+z0\min c^{\top} x + z_0 subject to Ax≤bA x \le b.

1.4 Integer Linear Programs

Integer linear programs are linear programs with the additional possible constraint that xi∈ℤx_i \in \mathbb{Z} for some chosen components xix_i.

This constraint gives us the power to express casework and conditional aspects of the optimization problem.

1.5 Convex Programs

An optimization problem is convex when

  1. We are minimizing the objective function.
  2. SS is a convex subset of ℝn\mathbb{R}^n.
  3. f:ℝn→ℝf: \mathbb{R}^n \to \mathbb{R} is convex on SS

Recall that we say that ff is convex on a convex set SS when for all x,y∈Sx, y \in S and any λ∈[0,1]\lambda \in [0,1] we have f(λx+(1−λ)y)≤λf(x)+(1−λ)f(y)f(\lambda x + (1 - \lambda) y) \le \lambda f(x) + (1 - \lambda) f(y). When −f-f is convex, we say that ff is concave.
