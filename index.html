Computing Setup

My hardware consists of a single laptop which I connect to an external monitor, keyboard, and mouse.

In pursuit of a simple, intuitive, and easily extensible computing experience, I use:

  • (cwm) as an X11 window manager, extended with custom scripts
  • (dmenu_run) as an application launcher
  • (dmenu) or (fzy) as a menu for user input in scripts
  • (kitty) as a terminal emulator, with (bash) integration
  • (kakoune) as a text editor, often with external programs like (figlet) and (boxes)
  • (firefox), (qutebrowser), or (lynx) as a web browser
  • (pipe-rename) or (mmv-go) for bulk renaming
  • (texmacs) occasionally for typesetting

For more information, take a look at my dotfiles [Github] and scripts [Github].
